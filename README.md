## 🚀 About KY AgentOps-IU

**KY AgentOps-IU**는 건양대학교의 AgentOps 기반 전공교육과정에서 생성되는  
AI Agent 산출물을 수집·관리·공유하기 위한 GitHub Organization입니다.

학생들은 단순히 AI Agent를 개발하는 것에 그치지 않고,  
AI Agent를 설계하고, 배포하고, 운영하고, 모니터링하며, 개선하는  
**AgentOps-IU(Improvement Unit)** 단위의 실무형 학습을 수행합니다.

---

## 🔁 What is AgentOps-IU?

**AgentOps-IU(Improvement Unit)** 는 AI Agent의 전주기 실습 단위입니다.

```text
Design → Build → Test → Deploy → Operate → Monitor → Improve → Share
```

하나의 IU는 단순 과제 제출물이 아니라,  
다음 학습과 후속 프로젝트의 출발점이 되는 **개선 단위**입니다.

---

## 🎯 Purpose

KY AgentOps-IU는 다음을 목표로 합니다.

| 목표 | 설명 |
|---|---|
| 산출물 축적 | 학생이 수행한 AI Agent 프로젝트를 표준 패키지로 관리 |
| 운영 경험 기록 | 운영문서, 로그, 평가 리포트, 개선 이력을 함께 보존 |
| 포트폴리오 공유 | 우수 산출물을 Portfolio Catalog 형태로 공개 |
| 교육 환류 | 이전 프로젝트를 Fork·개선하여 다음 AgentOps-IU로 확장 |
| 산학협력 연계 | 의료·국방·산업 분야 프로젝트를 기업 및 기관과 공유 |

## 📌 Contribution Guide

학생 프로젝트는 다음 원칙에 따라 등록합니다.

1. Repository는 표준 템플릿을 기반으로 생성합니다.
2. 민감 데이터, 개인정보, API Key는 업로드하지 않습니다.
3. 의료·국방 데이터 기반 프로젝트는 비식별화 자료만 공개합니다.
4. README에는 AgentOps-IU 수행 내역을 명확히 기록합니다.
5. 우수 산출물은 Portfolio Catalog에 등록합니다.
